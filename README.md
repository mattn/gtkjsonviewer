GtkJsonView
===========
A simple JSON viewer written in Python using GTK.

## Usage

### From a JSON file
```sh
python gtkjsonview.py test.json
```

### From an input stream
```sh
echo '[{"id":"1"}]' | python gtkjsonview.py
```

## Dependencies

* Python 3
* PyGObject module

Install:
```sh
sudo pip install PyGObject
```
