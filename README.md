GtkJsonView
===========
A simple JSON viewer written in GTK.

## Usage

### From a JSON file

    python gtkjsonview.py test.json

### From an input stream

    echo '[{"id":"1"}]' | python gtkjsonview.py

## Dependencies

* Python 3
* GTK module
* simplejson module

Install:

    sudo pip install simplejson
